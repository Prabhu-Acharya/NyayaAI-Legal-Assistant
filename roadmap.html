Week 7 — India Moat Features

India Moat = What no generic legaltech tool has
5 features. All India-specific. Each alone is a differentiator. Together = unfair advantage.
Hindi / Regional Language Contracts
User picks language. AI generates contract in Hindi (or Marathi/Tamil). Massive for Tier 2/3 users. Use Google Translate API or prompt Groq directly in Hindi.
BNS / IPC Section Citations
Inject BNS 2023 + IPC section refs into relevant contract clauses. e.g. NDA breach → BNS §111. Builds credibility. Add IPC→BNS mapping JSON.
Contract Risk Score (0–100)
After generation, Groq scores contract on 5 axes: enforceability, jurisdiction, ambiguity, IP protection, payment terms. Shows gauge UI. Premium upsell hook.
Voice Input (Speech-to-Text)
Web Speech API (free, Chrome). Mic button in chat + contract form. Hindi + English. 1 day of work. Competitor already has this.
WhatsApp Contract Delivery
Twilio WhatsApp API. After contract generated, user enters phone → gets PDF on WhatsApp. Indians live on WhatsApp. No competitor has this.

## Week 7 NyayaAI — India Moat Features

**Stack:** MERN (React 19 + Vite, Node/Express 5, MongoDB Atlas, Mongoose)
**AI:** Groq SDK, Llama 3.3 70B
**Auth:** JWT + bcrypt, authMiddleware.js
**Deploy:** Vercel (frontend) + Render (backend) + MongoDB Atlas
**Payments:** Razorpay ₹499/mo, isPremium on User model

**Done in Weeks 1–6:**
- Auth, AI Chat (ChatSession model), Contract Generator (5 types, PDF/DOCX, history)
- Payments + usage gating (3 free contracts/mo, monthly reset)
- helmet + rate-limit + express-validator
- Navbar, TopBar, ErrorBoundary, T&C Modal, Profile.jsx
- Document Analyzer: Multer → pdf2json/mammoth → Groq → risk score → report

**Week 7 — Build in order:**
1. Hindi/regional language contracts (language picker in ContractForm, Groq prompt in Hindi)
2. BNS/IPC citations in contracts (inject section refs from bns-map.json)
3. Contract risk score 0–100 (Groq scores on 5 axes, gauge UI)
4. Voice input — Web Speech API mic button in chat + contract form
5. WhatsApp PDF delivery — Twilio WhatsApp API after contract generated
